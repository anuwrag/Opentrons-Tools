Opentrons Protocol Designer JSON Editor

Description

This program allows you to modify the Opentrons Protocol Designer JSON file.
If you are observing bubbles, add push_out value = 0 if you are observing liquid inside the tip, add push_out value in range 1 - 10 for 1000 ul pipette.
