Protocol Run Time Analyzer

About This Tool

This tool helps you analyze Opentrons protocol run data by visualizing key information from protocol JSON files. Here's how to use it:

  1. Open the Opentrons App, click on Devices, and scroll down to view the recent protocol runs

  2. Click the "Choose File" button below and select your protocol JSON file
  3. The tool will automatically parse and display:
    • Basic protocol information (name, creation date, status)
    • Pipette configurations and their mounting positions
    • Labware locations and types
    • Runtime parameters used in the protocol
    • A detailed timeline of all commands with their execution times
  4. Use the command timeline table to analyze the duration of each step in your protocol
